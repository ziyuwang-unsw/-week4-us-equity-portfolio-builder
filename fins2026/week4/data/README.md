# Week 4 Data

This folder holds the small committed source inputs for the API week.

## Files

- `tiingo_intro_3.txt`
  - the warm-up ticker list for the first authenticated Tiingo example
  - default contents: `AAPL`, `MSFT`, `JPM`
- `tiingo_famous_50_pre2000.txt`
  - the larger 50-ticker Week 4 universe
  - intended for the final Tiingo pull that writes the long-format Parquet dataset
- `yahoo_intro_3.txt`
  - the warm-up ticker list for the first Yahoo Finance example
  - default contents: `AAPL`, `MSFT`, `JPM`
- `yahoo_famous_50.txt`
  - the larger 50-ticker Yahoo universe
  - intended for the final Yahoo pull that writes the long-format Parquet dataset
- `yahoo_app_10.txt`
  - the 10-stock Yahoo universe used by the Week 4 Streamlit app
  - includes Apple, Nvidia, Microsoft, and seven other large recognizable names

## Rules

- Keep only committed source inputs here.
- Save downloaded ECB, Kenneth French, Tiingo, or Yahoo outputs under `results/data/`.
- Never save API keys in this folder.
